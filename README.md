# 🪙 Conversor de Moedas Universal

Um conversor de moedas moderno, dinâmico e 100% responsivo, desenvolvido para entregar a melhor experiência de usuário na cotação de moedas em tempo real.

## 🚀 Funcionalidades

- **Cotações ao Vivo:** Integração com a AwesomeAPI para buscar os valores exatos de mercado no momento da conversão.
- **Máscara de Input Inteligente:** Formatação automática de valores monetários enquanto o usuário digita, ajustando o símbolo (R$, US$, €, £, ₿) de acordo com a moeda selecionada.
- **Conversão Flexível:** Permite converter de qualquer moeda para qualquer moeda disponível, não se limitando apenas ao Real como base.
- **Atalhos de Teclado:** Suporte à tecla `Enter` para acionar a conversão instantaneamente, melhorando a fluidez da navegação.
- **Design Responsivo:** Interface projetada com o conceito Mobile First, adaptando-se perfeitamente a celulares, tablets e monitores de qualquer tamanho.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização avançada, Media Queries para responsividade e variáveis de transição.
- **JavaScript (Vanilla):** Lógica de programação, manipulação do DOM, Event Listeners e consumo de API REST (Async/Await e Fetch).

## 💡 Como testar o projeto

Este é um projeto estático, ou seja, não precisa de instalações complexas para rodar.

1. Faça o clone deste repositório ou baixe os arquivos.
2. Extraia a pasta em seu computador.
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador padrão.
4. Digite um valor, escolha as moedas e teste as conversões!
